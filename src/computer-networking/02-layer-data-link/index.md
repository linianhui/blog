# 1 链路层的用途 {#data-link-purpose}

上一篇中说到物理层的用途是为上层提供`01`bit流，这个上层指的就是链路层。结合开篇中提到的计算机网络的核心设计理念**分组交换**，这一层的用途就是提供分组后的数据帧(`PDU=Frame`)。想要从下层的`01`比特中抽象出这样的结果，则需要解决一下的三个基本问题：

1. 封装成帧。
2. 透明传输。
3. 差错检测。
