# 1 概述 {#overview}

rust语言开发的开源的轻量级静态文件Http Server。支持的功能：
1. 静态文件目录服务。
2. 上传文件。
3. http基本认证。

```sh
miniserve --color-scheme squirrel --qrcode --auth lnh:123 --upload-files
```
